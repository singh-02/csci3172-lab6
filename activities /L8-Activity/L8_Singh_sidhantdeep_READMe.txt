Project Description
-----------------------------------
This exercise demonstrates basic DOM interaction, traversal, and manipulation.  
The calculator web page accepts two numeric inputs and performs four operations:  
• Addition  
• Subtraction  
• Multiplication  
• Division  

Results are shown dynamically on the page without reloading.  
The script also provides input validation and a friendly greeting message based on the time of day.

-----------------------------------
File List
-----------------------------------
index.html     – Main HTML structure and form UI  
styles.css     – Custom CSS styling (with normalize.css for reset)  
normalize.css – Normalize CSS defaults  
script.js     – JavaScript logic for DOM interaction and calculations  
Dal_Logo.png  – Header/logo image  
README.txt    – Project overview (this file)

-----------------------------------
Instructions to Run
-----------------------------------

1. Access the page via browser:
   https://web.cs.dal.ca/~yourcsusername/csci3172/activities/lecture8/

2. Enter two numbers and click any of the four buttons:
    Add  | Subtract | Multiply | Divide

3. The result will appear below the buttons instantly.
